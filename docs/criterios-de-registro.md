# Criterios de registro

Este documento orienta el registro de materiales, equipos y producciones dentro del Repositorio Accesible.

## Antes de registrar un recurso

Antes de incorporar un recurso, se recomienda verificar:

- Que el recurso tenga una finalidad educativa, accesible u organizativa clara.
- Que el nombre sea comprensible.
- Que la categoría sea adecuada.
- Que la descripción sea breve y útil.
- Que se conozca su procedencia, responsable o fuente.
- Que no se expongan datos personales sensibles.
- Que las imágenes utilizadas tengan autorización o sean referenciales.
- Que el recurso esté vigente o se indique si es referencial.

## Campos sugeridos

Cada recurso puede registrarse con los siguientes datos:

```text
Nombre del recurso:
Categoría:
Tipo de recurso:
Descripción breve:
Finalidad educativa:
Ubicación o procedencia:
Responsable o fuente:
Condición de disponibilidad:
Fecha de actualización:
Observaciones:
```

## Categorías sugeridas

- Materiales disponibles.
- Equipos tecnológicos.
- Materiales elaborados.
- Recursos adaptados.
- Producciones accesibles.
- Materiales de sensibilización.
- Recursos de apoyo docente o familiar.

## Accesibilidad del registro

Las descripciones deben ser claras, breves y comprensibles. Las imágenes deben tener texto alternativo y los enlaces deben indicar su destino o función.

## Actualización

Si un recurso deja de estar disponible, cambia de ubicación o requiere corrección, debe actualizarse el registro para evitar confusión.
