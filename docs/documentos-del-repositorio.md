# Documentos del repositorio

Este archivo lista los documentos de respaldo del repositorio Repositorio Accesible.

- `sustento-del-proyecto-personal.md`: documento central del proyecto personal.
- `autoria-y-contexto.md`: autoría, contexto de uso y reconocimiento.
- `fuentes-y-creditos.md`: fuentes, créditos y recursos externos o institucionales.
- `uso-permitido.md`: usos permitidos y condiciones generales.
- `alcance-del-repositorio.md`: alcance educativo, organizativo y funcional.
- `criterios-de-registro.md`: criterios para registrar materiales, equipos y producciones.

Estos documentos ayudan a conservar el orden, la finalidad educativa, la autoría, la trazabilidad y el uso responsable del repositorio.
